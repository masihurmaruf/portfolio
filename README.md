# Rails Portfolio App

## Overview

This is a portfolio app with Rails

## Project Information

The following was used to create this project

* Ruby 2.3.1
* Rails 5.0.1
* Bundler 1.13.6

## Getting started

* Checkout the project 
* Run `bundle install`
* Run `bundle exec rails server`
